Tech Word Finder

Algorithm A step-by-step set of instructions for a computer to follow, like a recipe for a cake.
Binary A way of counting using only two digits: 0 and 1. It’s the language computers speak!
Pixel Short for "picture element." These are the tiny dots of light that make up your screen.
Server A powerful computer that "serves" data to other computers, like a library for the internet.
